# Explaining Unsupervised Disease Staging in Huntington’s Disease

This repository provides explainability analyses to validate the proof-of-concept deployment of URL-STFN, demonstrating that the discovered disease stages and cluster structures correspond to clinically meaningful disease progression patterns in a subset of Enroll-HD patients.

The analyses investigate the learned representations, disease-stage clusters, and transition dynamics identified by URL-STFN using dimensionality reduction, saliency mapping, and SHAP-based feature attribution methods. The goal is to improve transparency, interpretability, and trust in unsupervised disease staging models for Huntington’s disease.
